# Vim
<leader> - Space

## Normal mode
<leader>w  -  save file
<leader>q  -  quit vim
<leader>n  -  new tab/buffer
<leader>.  -  next buffer
<leader>,  -  previouse buffer
<leader>vd -  jump to definition
<leader>i  -  Prettier
<leader>y  -  Global copy
<leader>p  -  Paste
<leader>f  -  Find word
<C-n> - NerdTree
<C-m> - Tagbar
gcgc  - toggle comment

## Visual mode
gc - comment/uncomment multiple lines
